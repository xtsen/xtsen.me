Standard

Le pack Standard est un pack idéal pour une gallerie, un portfolio ou encore une page de présentation. Ce pack ne contient pas beaucoup d'avantages mais est livré rapidement.

  • Avantages avec ce pack :
  • Nombre de pages : 1
  • Nombre de revisions : 2
  • Temps de livraison : 7

Pro

Le pack Pro est idéal pour un portfolio, un site vitrine ou encore un site pour une association. Ce pack a pour seul desavantage un nombre limité de page mais le temps de livraison reste tout de même très court.

  • Avantages avec ce pack :
  • Nombre de pages : 2-4
  • Nombre de revisions : 4
  • Temps de livraison : 10-15
  • Code source livré
  • Support de payment
  • Nom de domaine personalisé

Premium

Le pack Premium est parfait pour tout type de site. Il est autant adapté pour un site de dropshipping que pour un portfolio développé. Ce pack peut aussi être idéal pour un site tel que pour un site de documentation.

  • Avantages avec ce pack :
  • Nombre de pages : illimité
  • Nombre de revisions : 5
  • Temps de livraison : 15-30
  • Code source livré
  • Support de payment
  • Nom de domaine personalisé
